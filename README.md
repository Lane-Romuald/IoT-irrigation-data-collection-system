# IoT-irrigation-data-collection-system
Ce dépôt contient le code Arduino d'un système de collecte de données mesurant  les données sur l'humidité du sol, température du sol , humidité relative, température de l'air et probabilité de pluie. Le système est conçu pour recueillir des données environnementales en temps réel afin de faciliter l'irrigation intelligente et le suivi agricole.
